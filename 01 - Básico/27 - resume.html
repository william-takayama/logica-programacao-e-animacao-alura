CURSO - LÓGICA DE PROGRAMAÇÃO I

O que eu aprendi nesse curso de lógica de programação I?

As linguagens utilizadas foram: HTML + JavaScript

Sendo HTML uma linguagem estática e JavaScript uma linguagem dinâmica, ou seja, eu posso interagir com o usuário e criar desde alertas em POP-UPs até pegar informações digitadas pelo usuário.

O HTML é uma "linguagem" de marcação, ou seja, a partir de tags eu dou uma estrutura para o que desejo mostrar para meu usuário. Exemplos:

- Títulos de textos com importância para SEO


- Colocar links importantes, como: Clique aqui para acessar a W3Schools
- Aumentar uma fonte FONTE
- Arrumar problemas de acentuação com o meta charset ="UTF-8"!
